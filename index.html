Bachelor Thesis

Towards Automated Chemistry Research: Making Laboratory Devices Intelligent and Automation-Compatible — a lab automation proof-of-concept connecting a robot with a microplate reader to run measurements with minimal manual steps.

Demo
Load → measure → unload workflow demonstration.
Lab automation Robot integration Workflow design
Highlights
  • Goal: make lab devices easier to integrate into automated workflows.
  • Built: a working end-to-end pipeline where robot and instrument run in a coordinated sequence.
  • Focus: reliability (clear steps, synchronization, and safe execution).
  • Outcome: successful demo run showing the complete automation cycle.
  • Value: reduced manual handling and a structure that can be extended to other devices.
What I learned

In automation projects, clean sequencing and verification matter as much as motion control — small timing details decide if the system is reliable.

PolishBot — Erasing a drawing from the surface

Built an erase routine by first teaching a trajectory, then replaying it to clean a defined area reliably.

Punch points

  • Area to erase was defined by the user (not fixed in code).
